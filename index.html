Make.com Setup Steps
1
Create new scenario
In Make.com click "Create a new scenario" then press the "+" button in the center
2
Add Schedule trigger
Search "Schedule" → Every day → Set time to 03:00 UTC (= 7:00am UAE)
3
Add HTTP module for market data
Search "HTTP" → Make a request → Enter this URL:
https://www.alphavantage.co/query?function=BATCH_STOCK_QUOTES&symbols=NVDA,AMZN,MSFT,AAPL,GOOGL,MU&apikey=YOUR_KEY
4
Add Claude AI module
Add another HTTP module → POST to https://api.anthropic.com/v1/messages → Copy prompt from Claude Prompt tab
5
Add Telegram module
Search "Telegram Bot" → Send a Message → Enter your Chat ID → Map Claude's response as message
